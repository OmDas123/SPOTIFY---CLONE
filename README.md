# 🎵 Spotify Clone

A responsive web application that replicates the core user interface of the Spotify web player. This project was built entirely from scratch using vanilla web technologies to practice DOM manipulation, complex UI layouts, and native audio handling.

## ✨ Features

* **Audio Playback:** Controls to play, pause, skip tracks, and adjust volume using the native HTML5 Audio API.
* **Dynamic UI:** Responsive sidebar, playback bar, and main content area that mimic the official Spotify desktop experience.
* **Custom Styling:** Accurate recreation of Spotify's signature dark theme interface using pure CSS.
* **Interactive DOM:** Dynamic rendering of song lists and UI state updates using vanilla JavaScript.

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Flexbox / Grid)
* **Logic:** Vanilla JavaScript (ES6+)

## 🧠 What I Learned

Building this clone without frameworks was a major step up in my frontend journey. It specifically helped me improve my core skills in:
* Selecting, updating, and manipulating DOM elements efficiently.
* Managing event listeners for media controls (play, pause, progress bars).
* Structuring intricate web layouts using CSS Flexbox and Grid.
* Writing clean, modular JavaScript functions without relying on external libraries.
